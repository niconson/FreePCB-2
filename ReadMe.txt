FreePCB v2.4
--------------

В этой версии добавлена поддержка экспорта файлов в OpenSCAD — The Programmers Solid 3D CAD Modeller.
Проект собран на VS22. Если вы собираетесь компилировать проект у себя дома, то установите и выберите набор инструментов v100 !!! Если вы сумеете пересобрать решение на новейшей версии инструментов, то дайте знать об этом.
Актуальные задачи: пересобрать проект с использованием новейшей версии инструментов Visual Studio. Возможные варианты решения данной задачи:
- сделать библиотеку clibpdf.lib в виде DLL файла
- или перейти на другую библиотеку экспорта PDF

Присоединяйтесь к разработке этого опенсоурс проекта !
